# Brain-Boost

A modern, futuristic, and interactive front-end website featuring animated UI elements, a 3D rotating tag cloud, smooth navigation, and responsive mobile design. Built with **HTML**, **CSS**, and **Vanilla JavaScript**.

---

## Features

- **3D Rotating Tag Cloud**
  - Interactive technology tags (JavaScript, CSS, HTML, Python, React, etc.)
  - Clickable tags that open course pages in a new tab
- **Futuristic UI & Animations**
  - Neon glow effects
  - Hover animations, glitch effects, fade-in sections
  - Cursor glow effect on desktop
- **Responsive Mobile Menu**
  - Animated slide-in navigation
  - Touch-friendly and keyboard accessible
- **Smooth Scrolling**
  - Animated scroll to sections and back-to-top support
- **Student Application Form**
  - Client-side validation
  - Animated submit feedback (no backend required)
- **Custom Select Dropdown**
  - Accessible, keyboard-navigable faculty selector
- **Performance Optimized**
  - Lightweight JavaScript
  - Throttled mouse movement effects

## Getting Started

1. Download or clone the repository
2. Open `index.html` in your browser
3. No build tools or server required

---

## Tag Cloud Navigation

Each tag opens a related course page inside the `kurset/` directory.

Example:
- JavaScript → `kurset/javascript.html`
- CSS → `kurset/css.html`
- HTML → `kurset/HTML.html`
- Python → `kurset/python.html`
