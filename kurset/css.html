Brain Boost - Kursi CSS

Modulet e Kursit

1. Hyrje në CSS

Mësoni bazat e CSS: si funksionon stilizimi i faqeve web, çfarë janë selektorët dhe si lidhet CSS me HTML.


p {
    color: blue;
    background-color: aquamarine;
}

2. Inline, Internal dhe External CSS

CSS mund të aplikohet në tre mënyra kryesore:

  • Inline CSS: aplikohet direkt brenda elementit HTML për ndryshime të shpejta.
  • Internal CSS: vendoset në tagun <style> brenda <head> të një faqe HTML.
  • External CSS: vendoset në një file të veçantë .css dhe lidhet me HTML përmes tagut <link>.

Këto metoda ndihmojnë në menaxhimin e stileve sipas madhësisë dhe kompleksitetit të projektit.

3. Selektorët në CSS

Selektorët zgjedhin elementet që do të stilizohen.


h1 { color: yellow; }


.box { background: #222; padding: 20px; }


#unik { color: cyan; }

4. Ngjyrat, Background dhe Border

CSS ofron shumë mënyra për të modifikuar pamjen e elementeve.

.box {
    background-color: #1a1a1a;
    border: 2px solid #ffcc00;
    color: white;
    padding: 15px;
}

5. Layout: Flexbox & Grid

Mësoni dy sistemet më të fuqishme për ndërtimin e faqeve moderne.


.container {
    display: flex;
    justify-content: center;
    gap: 20px;
}


.gridi {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 15px;
}

6. Animacionet dhe Transition

Si t'i japim lëvizje dhe efekte elementeve.

.box:hover {
    transform: scale(1.1);
    transition: 0.3s;
}

@keyframes rrotullim {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
}

7. Shembuj praktik me CSS

Krijimi i një butoni modern, një kartele dhe një paneli navigimi.

button {
    background: #00bfff;
    padding: 12px 20px;
    border-radius: 8px;
    border: none;
    color: white;
    font-size: 16px;
    transition: 0.3s;
}
button:hover {
    background: #0088cc;
}
