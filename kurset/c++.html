Brain Boost - Kursi C++

Modulet e Kursit

1. Hyrje në C++

Çfarë është C++, pse përdoret dhe ku aplikohet (lojëra, sisteme, algoritme, embedded).

// Programi më i thjeshtë në C++
#include <iostream>
using namespace std;

int main() {
    cout << "Hello C++!" << endl;
    return 0;
}

2. Variablat dhe Tipet e të Dhënave

Si të krijojmë variabla në C++.

int mosha = 19;
double paga = 450.50;
char shkronja = 'A';
bool aktiv = true;
string emri = "Arlinda";

3. Input / Output

Si të marrim të dhëna nga përdoruesi dhe t'i shfaqim.

#include <iostream>
using namespace std;

int main() {
    int mosha;
    cout << "Shkruaj moshen: ";
    cin >> mosha;

    cout << "Mosha juaj eshte: " << mosha << endl;
}

4. If / Else – Kushtet

Vendimmarrja në C++.

int mosha = 18;

if (mosha >= 18) {
    cout << "Je i rritur";
} else {
    cout << "Je i mitur";
}

5. Cikli For

Përdorimi i cikleve.

for (int i = 1; i <= 5; i++) {
    cout << "Numri: " << i << endl;
}

6. Funksionet

Si të krijoni funksione të thjeshta në C++.

int shuma(int a, int b) {
    return a + b;
}

cout << shuma(4, 7); // 11

7. Arrays

Ruajtja e shumë vlerave në një variabël.

int nota[5] = {10, 9, 8, 9, 10};

for (int i = 0; i < 5; i++) {
    cout << nota[i] << " ";
}

8. String-et në C++

Punimi me tekste.

string emri = "Arlinda";
cout << "Gjatësia: " << emri.length();

// Bashkim string-ësh
string a = "Hello ";
string b = "C++";
cout << a + b;

9. OOP – Programimi i Orientuar në Objekte

Klasat dhe objektet – baza e C++.

class Student {
public:
    string emri;
    int mosha;

    void prezanto() {
        cout << "Une jam " << emri << ", mosha " << mosha;
    }
};

int main() {
    Student s;
    s.emri = "Arlinda";
    s.mosha = 19;
    s.prezanto();
}

10. Konstruktorët

Funksionet që ekzekutohen automatikisht kur krijohet objekti.

class Vetura {
public:
    string modeli;

    // Konstruktor
    Vetura(string m) {
        modeli = m;
    }
};

int main() {
    Vetura v("BMW");
    cout << v.modeli;
}

11. Pointerët

Një prej koncepteve më të rëndësishme në C++.

int x = 10;
int* ptr = &x;

cout << ptr;   // adresa
cout << *ptr;  // vlera 10

12. File Handling – Shkrimi & Leximi në File

Si të punoni me file në C++.

#include <fstream>

ofstream file("test.txt");
file << "Pershendetje!";
file.close();
