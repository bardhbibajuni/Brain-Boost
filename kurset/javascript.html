Brain Boost - Kursi JavaScript

Modulet e Kursit

1. Hyrje në JavaScript

JavaScript është gjuha kryesore e web-it. Ky modul shpjegon çfarë është JavaScript, ku përdoret dhe si ekzekutohet në browser. Kështu do të krijoni skriptin tuaj të parë.

// Programi i parë në JavaScript
console.log("Mirë se vini në JavaScript!");

2. Variablat dhe tipet e të dhënave

Do të mësoni dallimin mes var, let dhe const, si funksionojnë stringjet, numrat, bool-et dhe tipet e tjera të thjeshta.

let emri="Ardita";
const mosha=21;

console.log(emri + " ka " + mosha + " vjet.");

let pi=3.1415;
console.log("Vlera e pi është: " + pi);

3. Kushtet dhe ciklet

JavaScript përdor if/else, switch, for, while dhe struktura të tjera për kontrollin e rrjedhës së kodit.

let numri=5;

if (numri>0) {
    console.log("Pozitiv");
} else {
    console.log("Negativ");
}

for (let i=0;i<5;i++) {
    console.log("Numri: "+ i);
}

let shuma=0;
let n=0;

while (n<5) {
    shuma+= n;
    n++;
}

console.log("Shuma: " + shuma);

4. Funksionet

Funksionet janë një pjesë thelbësore e JavaScript. Do të shihni funksione normale, arrow functions dhe funksione për kthimin e vlerave.

function pershendetje() {
    console.log("Përshëndetje!");
}

const shuma = (a,b) => {
    return a+b;
};

pershendetje();
console.log(shuma(5,3));

5. Arrays dhe Objects

JavaScript është shumë fleksibël me strukturat e të dhënave. Do të përdorni arrays, objects dhe metoda të ndryshme për manipulimin e tyre.

let frutat = ["Mollë","Banane","Dardhë"];

frutat.forEach(f =>console.log(f));

let person={
    emri: "Ardita",
    mosha: 21
};

console.log(person.emri);

6. DOM-Manipulimi i elementeve

DOM ju lejon të ndryshoni HTML dhe CSS me JavaScript. Ky modul ju mëson të krijoni, lexoni dhe manipuloni elemente.

// Ndërro tekstin e një elementi
document.getElementById("titulli").innerText = "Teksti i ri";

// Krijo element të ri
let p=document.createElement("p");
p.innerText="Ky paragraf u shtua me JS";
document.body.appendChild(p);

Në fund do të zbatoni gjithçka që keni mësuar duke ndërtuar mini-projekte.

// Shembull: Trego nëse një numër është çift apo tek

if (num%2===0) {
    console.log("Çift");
