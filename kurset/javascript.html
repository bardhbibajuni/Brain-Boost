Brain Boost - Kursi i JavaScript

Modulet e Kursit

1. Hyrje në JavaScript

JavaScript është gjuha kryesore për ndërtimin e faqeve dinamike. Ajo ekzekutohet në shfletues dhe mund të modifikojë HTML, CSS, të dërgojë kërkesa në server dhe të krijojë aplikacione interaktive.

Çfarë mund të bëni me JavaScript?

  • Të ndryshoni përmbajtjen e faqes pa e rifreskuar
  • Të validoni forma
  • Të krijoni animacione dhe galeri
  • Të komunikoni me serverin përmes AJAX / Fetch API
console.log("Mirësevini në JavaScript!");
alert("JavaScript po funksionon!");

2. Variablat dhe tipet e të dhënave

JavaScript ka tre mënyra kryesore deklarimi: var, let dhe const. Rekomandohet të përdorni let dhe const.

Tipet primitive: string, number, boolean, null, undefined, bigint, symbol.

let emri = "Ardita";
const mosha = 21;
let nota = 9.5;
console.log(`${emri} ka notën ${nota}`);

let booleanTest = true;
console.log(typeof booleanTest);

3. Kushtet dhe ciklet

Kushtet kontrollojnë rrjedhën e programit. Ciklet përsërisin logjikën disa herë.

Kushtet kryesore: if, else if, else, switch.

Ciklet: for, while, do-while.

let numri = 7;
if (numri % 2 === 0) console.log("Çift");
else console.log("Tek");

for (let i = 1; i <= 5; i++) console.log("Numri: " + i);

let x = 0;
do {
    console.log("x = " + x);
    x++;
} while (x < 3);

4. Funksionet

Funksionet organizojnë kodin dhe e bëjnë atë të ripërdorshëm. Mund të jenë klasike ose funksione shigjetë.

Funksione shigjetë: janë më të shkurtra dhe moderne.

function pershendet(emri) {
    console.log("Përshëndetje " + emri);
}

const shuma = (a, b) => a + b;

pershendet("Ardita");
console.log(shuma(5, 4));

5. Arrays dhe Objects

Arrays ruhen lista vlerash, ndërsa objects ruhajnë të dhëna të strukturuara me çelësa dhe vlera.

Metoda të dobishme të array: map(), filter(), forEach(), reduce().

let frutat = ["Mollë", "Banane", "Kivi"];
frutat.map(f => console.log(f));

let student = {
    emri: "Ardita",
    mosha: 21,
    drejtimi: "Shkenca Kompjuterike"
};
console.log(student.drejtimi);

6. DOM - Manipulimi i faqes web

DOM ju lejon të ndryshoni elementet HTML në kohë reale.

Metoda kryesore: getElementById, querySelector, classList, innerText, innerHTML.

let titulli = document.querySelector("#titulli");
titulli.innerText = "Titulli u ndryshua!";

document.body.style.backgroundColor = "#222";

7. Eventet

Eventet ju lejojnë të reagoni ndaj ndërveprimeve të përdoruesit.

Shembuj events: click, mouseover, keyup, input.

const btn = document.getElementById("butoni");
btn.addEventListener("click", () => {
    alert("U klikua!");
});

8. Projekte praktike

Këto janë ushtrime të vogla për të forcuar njohuritë.

// Kalkulator i thjeshtë
function mbledhja(a, b) { return a + b; }
console.log(mbledhja(10, 5));

// Numërimi i shkronjave
let fjala = "JavaScript";
console.log(fjala.length);

// Filtrimi i numrave
let lista = [1,2,3,4,5,6];
let cift = lista.filter(n => n % 2 === 0);
console.log(cift);
