Modulet e Kursit

1. Hyrje në Python

Ky modul ju prezanton me Python në mënyrë të thjeshtë dhe të qartë...

# Kontrollo versionin
python --version

# Printim i parë
print("Mirë se vini në Python!")

2. Sintaksa bazë dhe variabla

Në këtë modul do të njiheni me bazat e Python-it...

emri = "Ardita"
mosha = 21
print(f"{emri} ka {mosha} vjet.")

pi = 3.1415
print("Vlera e pi:", pi)

3. Kushtet dhe ciklet

Ky modul ju ndihmon të përdorni kushtet dhe ciklet...

numri = 5
if numri > 0:
    print("Pozitiv")
else:
    print("Jo pozitiv")

4. Funksionet

Funksionet e bëjnë programin më të organizuar...

def pershendetje():
    print("Përshëndetje!")

5. Lista dhe koleksione të tjera

Ky modul ju mëson si të përdorni strukturat kryesore...

frutat = ["mollë", "banane", "dardhë"]
frutat.append("portokall")
for fruta in frutat:
    print(fruta)

6. Modulet dhe libraritë standarde

Modulet dhe libraritë standarde ju mundësojnë të përdorni funksione të gatshme...

import math
import random
print(math.sqrt(25))
print(random.randint(1, 10))

7. Projekte praktike

Në këtë modul do të praktikoni njohuritë e fituara duke krijuar projekte të vogla...

numri = 8
if numri % 2 == 0:
    print("Çift")
else:
    print("Tek")
