Modulet e Kursit

1. Hyrje në C

Çfarë është gjuha C, historiku, pse përdoret dhe ku aplikohet (OS, embedded, sisteme, algoritme).

// Programi më i thjeshtë në C
#include <stdio.h>

int main() {
    printf("Hello C!");
    return 0;
}

2. Variablat dhe Tipet e të Dhënave

Tipet bazë të të dhënave dhe deklarimi i variablave në C.

int mosha = 20;
float nota = 9.5;
char shkronja = 'A';
double paga = 450.75;

3. Input & Output me printf dhe scanf

Leximi i të dhënave nga përdoruesi.

#include <stdio.h>

int main() {
    int mosha;
    printf("Shkruaj moshen: ");
    scanf("%d", &mosha);

    printf("Mosha juaj eshte: %d", mosha);
}

4. If / Else - Kushtet në C

Kushtet bazike me sintaksën e C.

int nota = 8;

if (nota >= 5) {
    printf("Kalove!");
} else {
    printf("Nuk kalove.");
}

5. Cikli For, While, Do-While

Përsëritja e veprimeve.

// For
for (int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}

// While
int x = 1;
while (x <= 5) {
    printf("%d\n", x);
    x++;
}

6. Funksionet

Krijimi i funksioneve dhe thirrja e tyre.

int shuma(int a, int b) {
    return a + b;
}

int main() {
    printf("%d", shuma(4, 6)); // 10
}

7. Arrays në C

Punimi me lista të dhënash.

int nota[5] = {10, 9, 8, 7, 10};

for (int i = 0; i < 5; i++) {
    printf("%d ", nota[i]);
}

8. String-et në C

String-et janë arrays të tipit char.

char emri[] = "Arlinda";

printf("Emri: %s\n", emri);
printf("Shkronja e pare: %c\n", emri[0]);

9. Pointerët në C

Një prej temave më të rëndësishme dhe fondamente e C.

int x = 10;
int* ptr = &x;

printf("Adresa: %p\n", ptr);
printf("Vlera: %d", *ptr);

10. Struktura (struct)

Si të krijojmë tipe të personalizuara të të dhënave.

struct Student {
    char emri[20];
    int mosha;
};

int main() {
    struct Student s = {"Arlinda", 19};
    printf("%s - %d", s.emri, s.mosha);
}

11. Dynamic Memory - malloc, free

Alokimi dinamik i memories.

#include <stdlib.h>

int* ptr = malloc(sizeof(int));
*ptr = 25;

printf("%d", *ptr);

free(ptr);

12. File Handling - Lexim & Shkrim në File

Punimi me file në C.

#include <stdio.h>

int main() {
    FILE *file = fopen("test.txt", "w");
    fprintf(file, "Pershendetje!");
    fclose(file);
}
