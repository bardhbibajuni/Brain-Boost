Brain Boost - Kursi Node.js

Modulet e Kursit

1. Hyrje në Node.js

Çfarë është Node.js dhe pse përdoret për backend development.

// Kontrollo versionin e Node
node -v

// Kontrollo versionin e npm
npm -v

2. Krijimi i projektit

Si të krijoni një projekt të ri Node.js me npm.

mkdir projekti-im
cd projekti-im

npm init -y

3. Modules në Node.js (require / export)

Node përdor module për të organizuar kodin.

// file: math.js
function shuma(a, b) {
  return a + b;
}
module.exports = shuma;

// file: app.js
const shuma = require('./math');
console.log(shuma(5, 7));

4. Krijimi i një serveri HTTP

Serveri më bazik në Node.js.

const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello nga Node.js!');
});

server.listen(3000, () => {
  console.log('Serveri po punon në portin 3000');
});

5. fs module – Leximi dhe shkrimi i fajllave

Si të lexoni dhe shkruani fajlla me Node.

const fs = require('fs');

fs.writeFileSync('test.txt', 'Pershendetje!');
const data = fs.readFileSync('test.txt', 'utf8');

console.log(data);

6. npm – Instalimi i paketave

Si të instaloni dhe përdorni paketa të jashtme.

npm install axios

const axios = require('axios');

axios.get('https://api.example.com')
  .then(res => console.log(res.data));

7. Express.js – Framework për servera

Express është framework-u më i përdorur për Node.js servers.

npm install express

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello nga Express!');
});

app.listen(3000, () => console.log('Serveri Express po punon!'));

8. Middleware në Express

Si të përdorni middleware për logim, validime etj.

app.use((req, res, next) => {
  console.log('Një kërkesë u pranua!');
  next();
});

9. Routes (Rrugët)

Si të krijoni ruterë të ndarë për endpoint-e të ndryshme.

// file: routes/studentet.js
const express = require('express');
const router = express.Router();

router.get('/', (req, res) => {
  res.send('Lista e studenteve');
});

module.exports = router;

// app.js
const studentetRoute = require('./routes/studentet');
app.use('/studentet', studentetRoute);

10. Async / Await në Node.js

Menaxhimi i operacioneve async.

function prit(ms) {
  return new Promise(resolve => setTimeout(resolve, ms));
}

async function demo() {
  console.log('Duke pritur...');
  await prit(2000);
  console.log('Mbaroi!');
}

demo();

11. Lidhja me MySQL

Node.js mund të lidhet me MySQL shumë thjeshtë.

npm install mysql2

const mysql = require('mysql2');

const db = mysql.createConnection({
  host: 'localhost',
  user: 'root',
  password: '',
  database: 'shkolla'
});

db.connect(err => {
  if (err) throw err;
  console.log('Lidhja me MySQL u realizua!');
});

12. API me Express + MySQL

Krijimi i një API të thjeshtë.

app.get('/studentet', (req, res) => {
  db.query('SELECT * FROM studentet', (err, results) => {
    if (err) throw err;
    res.json(results);
  });
});
