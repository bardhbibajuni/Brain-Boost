Modulet e Kursit

1. Hyrje në Java

Ky modul ju prezanton me Java dhe konceptet bazë të programimit të orientuar ne objekte. Do të mësoni pse Java është një gjuhë e fuqishme dhe e përdorur gjerësisht, si të instaloni JDK dhe si të krijoni programin tuaj të parë.

// Programi i parë në Java
public class Main {
    public static void main(String[] args) {
        System.out.println("Mirë se vini në Java!");
    }
}

2. Sintaksa bazë dhe variabla

Në këtë modul do të mësoni bazat e Java: variablat, llojet e të dhënave, operatorët dhe mënyrën e printimit në ekran. Kjo do t'ju ndihmojë të krijoni programe të thjeshta dhe të kuptoni strukturën e kodit.

String emri = "Ardita";
int mosha = 21;
System.out.println(emri + " ka " + mosha + " vjet.");

double pi = 3.1415;
System.out.println("Vlera e pi: " + pi);

3. Kushtet dhe ciklet

Ky modul ju ndihmon të përdorni kushtet if/else dhe ciklet for/while për të kontrolluar rrjedhën e programit. Do të praktikoni mënyra të ndryshme për të përsëritur veprime dhe për të zgjidhur probleme të thjeshta logjike.

int numri = 5;
if (numri > 0) {
    System.out.println("Pozitiv");
} else {
    System.out.println("Jo pozitiv");
}

for (int i = 0; i < 5; i++) {
    System.out.println("Numri është: " + i);
}

int shuma = 0;
int n = 0;
while (n < 5) {
    shuma += n;
    n++;
}
System.out.println("Shuma e numrave: " + shuma);

4. Funksionet dhe Metodat

Funksionet (metodat) i bëjnë programet më të organizuara dhe ripërdorshme. Në këtë modul do të mësoni si të krijoni metoda të thjeshta me dhe pa parametra, si të ktheni vlera dhe si t'i përdorni për të strukturuar programin tuaj.

public class Main {
    public static void pershendetje() {
        System.out.println("Përshëndetje!");
    }

    public static int shuma(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        pershendetje();
        System.out.println(shuma(5, 3));
    }
}

5. Arrays dhe koleksione të tjera

Ky modul ju mëson strukturat bazë të të dhënave në Java: arrays, ArrayList dhe HashMap. Do të praktikoni manipulimin, shtimin dhe leximin e elementeve për të menaxhuar informacionin brenda programeve tuaja. 

String[] frutat = {"Mollë", "Banane", "Dardhë"};
for(String fruta : frutat) {
    System.out.println(fruta);
}

import java.util.ArrayList;
ArrayList lista = new ArrayList<>();
lista.add("Portokall");
for(String item : lista) {
    System.out.println(item);
}

6. Libraritë standarde dhe paketa

Libraritë standarde në Java ofrojnë metoda të gatshme për manipulim teksti, llogaritje matematikore dhe punë me koleksione. Ky modul ju ndihmon të kurseni kohë dhe të bëni programim më efikas duke përdorur klasa dhe paketa të gatshme.

import java.util.Random;

Random rand = new Random();
System.out.println("Numër rastësor: " + rand.nextInt(10));

System.out.println("Vlera e sqrt(25): " + Math.sqrt(25));

7. Projekte praktike

Në këtë modul do të praktikoni gjithçka të mësuar duke krijuar projekte të vogla dhe funksionale. Kjo do t'ju ndihmojë të aplikoni teorinë në probleme reale, duke përdorur cikle, metoda dhe koleksione për të zgjidhur probleme të ndryshme.

int numri = 8;
if(numri % 2 == 0) {
    System.out.println("Çift");
} else {
    System.out.println("Tek");
}

// Llogarit shumen e një array
int[] numrat = {1,2,3,4};
int shuma = 0;
for(int n : numrat) {
    shuma += n;
}
System.out.println("Shuma e array: " + shuma);
