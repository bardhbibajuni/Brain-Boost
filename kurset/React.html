Modulet e Kursit

1. Hyrje në React

Çfarë është React dhe pse përdoret për frontend development.

// Kontrollo versionin e Node (nevojitet për React)
node -v

// Kontrollo versionin e npm
npm -v

2. Krijimi i projektit me Create React App

Si të krijoni një projekt të ri React me npm.

npx create-react-app projekti-im
cd projekti-im
npm start

3. Komponentët në React

React përdor komponentë për të ndarë UI-në në pjesë të ripërdorshme.

// file: App.js
function App() {
  return <h1>Hello nga React!</h1>;
}

export default App;

4. JSX - Sintaksa e React

Si të shkruani HTML brenda JavaScript.

const element = <h1>Pershendetje, React!</h1>;

5. Props - Parametrat e komponentëve

Si të kaloni të dhëna tek komponentët.

function Welcome(props) {
  return <h1>Pershendetje, {props.emri}!</h1>;
}

<Welcome emri="Arlinda" />

6. State - Gjendja e komponentëve

Si të menaxhoni gjendjen brenda komponentëve.

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  return (
    <div>
      <p>Numri: {count}</p>
      <button onClick={() => setCount(count + 1)}>Rrit</button>
    </div>
  );
}

7. useEffect - Efektet anësore

Si të bëni kërkesa HTTP ose efekte të tjera anësore.

import { useEffect, useState } from 'react';
import axios from 'axios';

function DataFetcher() {
  const [data, setData] = useState(null);

  useEffect(() => {
    axios.get('https://api.example.com')
      .then(res => setData(res.data));
  }, []);

  return <div>{JSON.stringify(data)}</div>;
}

8. Events - Ngjarjet

Si të trajtoni ngjarje në React.

function Button() {
  const handleClick = () => {
    alert('U klikua butoni!');
  }

  return <button onClick={handleClick}>Kliko</button>;
}

9. Ruterët me React Router

Si të krijoni ruterë për faqet e ndryshme.

npm install react-router-dom

import { BrowserRouter, Routes, Route } from 'react-router-dom';
import Home from './Home';
import About from './About';

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </BrowserRouter>
  );
}

10. Conditional Rendering

Si të shfaqni komponentë në bazë të kushteve.

function Greeting({ loggedIn }) {
  return (
    <div>
      {loggedIn ? <p>Mirë se erdhët!</p> : <p>Ju lutem kyçuni.</p>}
    </div>
  );
}

11. Lidhja me API

React mund të lidhet me API shumë lehtë.

import { useEffect, useState } from 'react';
import axios from 'axios';

function Studentet() {
  const [studentet, setStudentet] = useState([]);

  useEffect(() => {
    axios.get('/api/studentet')
      .then(res => setStudentet(res.data));
  }, []);

  return (
    <ul>
      {studentet.map(s => <li key={s.id}>{s.emri}</li>)}
    </ul>
  );
}

12. Forme dhe Controlled Components

Si të krijoni forma dhe të lexoni inputet e përdoruesit.

import { useState } from 'react';

function KontaktForm() {
  const [emri, setEmri] = useState('');

  const handleSubmit = (e) => {
    e.preventDefault();
    alert(`Faleminderit, ${emri}!`);
  }

  return (
    <form onSubmit={handleSubmit}>
      <input type="text" value={emri} onChange={e => setEmri(e.target.value)} />
      <button type="submit">Dergo</button>
    </form>
  );
}
